Tag 2 bauen das Frontend — Bun + React + Tailwind + TypeScript. Du wirst dann zum ersten Mal deine Pflanzen im Browser sehen 🌿

cd ~/Antonio/Projekt/shopdrop/frontend
bun create vite . --template react-ts

bun install
bun add -d tailwindcss @tailwindcss/vite
Jetzt Tailwind konfigurieren. Öffne vite.config.ts und füge in zeile pluging tailwindcss und import tailwindcss from '@tailwindcss/vite' ein.

Dann src/index.css — alles löschen und ersetzen mit:
@import "tailwindcss";

bun run dev dann sieht ➜  Local: http://localhost:5173/
