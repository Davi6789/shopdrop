Schritt 1 — Backend: POST /orders Route — nimmt Name, Adresse und Warenkorb entgegen, speichert alles in orders + order_items, reduziert den stock der Produkte.
- Erstelle backend/src/routes/orders.ts

- Jetzt die Route in backend/src/index.ts einbinden 
    import ordersRouter from './routes/orders';   // ← neu

    app.use('/orders', ordersRouter);              // ← neu

- Route testen — in einem neuen Terminal:
    curl -X POST http://localhost:3001/orders \
  -H "Content-Type: application/json" \
  -d '{
    "customer_name": "Antonio Test",
    "customer_address": "Musterstraße 1, Berlin",
    "items": [
      { "product_id": 1, "quantity": 1 },
      { "product_id": 2, "quantity": 2 }
    ]
  }'

  prüfen ob der Stock sich reduziert hat
  psql shopdrop -c "SELECT id, name, stock FROM products WHERE id IN (1,2);"
